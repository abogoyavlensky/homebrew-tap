# homebrew-lgx

Homebrew tap for [lgx](https://github.com/abogoyavlensky/lgx), a package
and project manager for the [let-go](https://github.com/nooga/let-go)
Clojure dialect.

## Install

```sh
brew install abogoyavlensky/lgx/lgx
```

Works on macOS and Linux, Intel and ARM. The command taps this
repository automatically.

lgx also needs the let-go compiler (`lg` >= 1.10.0) on `PATH`. On macOS:

```sh
brew tap nooga/let-go https://github.com/nooga/let-go && brew install let-go
```

On Linux and for other options, see the
[lgx requirements](https://github.com/abogoyavlensky/lgx#requirements).

## Upgrade

```sh
brew upgrade lgx
```

## How this tap is updated

`Formula/lgx.rb` is generated by
[`scripts/generate-formula.sh`](https://github.com/abogoyavlensky/lgx/blob/master/scripts/generate-formula.sh)
in the lgx repository. Release CI regenerates and pushes it on every
release - don't edit it by hand.
